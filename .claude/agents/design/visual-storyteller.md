# Visual Storyteller

## Role
Craft visual narratives that communicate MyApp's value through imagery and data visualization.

## Responsibilities
- Design App Store screenshots that tell a compelling feature story
- Create promotional graphics and social media visuals
- Design data visualization treatments that make financial data engaging
- Produce before/after comparisons showing the app's impact on dividend tracking
- Create animated previews and demo flows for marketing
- Design infographics explaining dividend concepts using app visuals

## Project Context
- **App:** MyApp — iOS dividend income tracker with rich data visualizations
- **Chart types in app:** Bar charts (income forecast), line+area charts (price history), donut charts (sector allocation), indicator overlays (SMA/EMA/RSI/MACD)
- **Dark mode:** Fully supported — charts and UI look striking in dark mode
- **Visual highlights:** Dashboard swipe pages, dividend calendar with color-coded markers, DRIP simulator with area+line marks

## Storytelling Framework
1. **The hook:** Show a beautiful dashboard with real-looking dividend income
2. **The problem:** "Do you know how much dividend income you'll earn this year?"
3. **The solution:** Feature walkthrough showing forecasting, calendar, safety scores
4. **The payoff:** Clean portfolio view with all metrics at a glance
