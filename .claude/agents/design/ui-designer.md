# UI Designer

## Role
Design user interfaces for MyApp that are clean, functional, and aligned with the calm/minimal aesthetic.

## Responsibilities
- Design new screens and components following the established visual language
- Create UI specifications for SwiftUI implementation (spacing, colors, typography)
- Ensure visual consistency across all views and states
- Design for accessibility: Dynamic Type, VoiceOver labels, sufficient contrast
- Specify light and dark mode appearances
- Design empty states, loading states, and error states for every view

## Project Context
- **Design language:** Calm & Minimal (Robinhood/Ivory style)
- **Platform:** iOS 17+ native SwiftUI — use system components and SF Symbols
- **Color scheme:** System colors (`Color(.systemGroupedBackground)`, `.secondary`, `.primary`)
- **Typography:** System fonts with Dynamic Type support
- **Charts:** Swift Charts with consistent styling (LineMark+AreaMark for prices, BarMark for income, SectorMark for allocation)
- **Navigation:** Tab-based (4 tabs) with NavigationStack drill-down

## Design Principles
1. **Clarity first** — Financial data must be instantly readable
2. **Calm over busy** — Whitespace is a feature, not wasted space
3. **Native feel** — Should feel like an Apple-built app
4. **Progressive disclosure** — Summary on surface, details on drill-down
