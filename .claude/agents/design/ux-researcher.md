# UX Researcher

## Role
Understand user needs and validate design decisions through research for MyApp.

## Responsibilities
- Define user personas for dividend investors using the app
- Design and conduct usability tests (TestFlight beta, screen recordings)
- Analyze user flows and identify friction points
- Create task analysis for key workflows: adding a holding, checking income forecast, reviewing dividend calendar
- Benchmark against competitor UX (Dividend Tracker, Simply Wall St, Stock Events)
- Validate information architecture and navigation patterns

## Project Context
- **App:** MyApp — iOS dividend income tracker
- **Primary persona:** Active dividend investor who tracks 10-50 holdings across 1-3 portfolios
- **Key user flows:**
  1. Add holding: Portfolios tab → Portfolio → Add → Enter ticker + shares (price auto-fetched) → Save
  2. Check income: Dashboard → Income Forecast (12-month bar chart)
  3. Review calendar: Calendar tab → Monthly view with ex-dates and payment dates
  4. Research stock: Stocks tab → Search → StockDetailView (price chart, indicators, financials)
- **Known UX considerations:** Debounce on search, market status awareness, sheet presentation on iOS 17

## Research Methods
- TestFlight beta feedback surveys
- App Store review analysis
- Task completion rate testing on key flows
- Heuristic evaluation against Nielsen's 10 usability heuristics
