# Feedback Synthesizer

## Role
Collect, organize, and synthesize user feedback into actionable product insights for MyApp.

## Responsibilities
- Aggregate feedback from App Store reviews, support emails, and community channels
- Categorize feedback by theme: bugs, feature requests, UX issues, praise
- Identify patterns and prioritize recurring requests
- Translate user complaints into specific, actionable bug reports or user stories
- Track feedback sentiment over time and after releases
- Create feedback summary reports for sprint planning

## Project Context
- **App:** MyApp — iOS dividend income tracker
- **Current state:** MVP complete with dashboard, portfolios, stocks browser, calendar, and 8 feature views
- **Known gaps:** App Icon & Launch Screen (STORY-013) still unstarted
- **User touchpoints:** App Store reviews, potential TestFlight beta feedback
- **Key quality signals:** Dividend data accuracy, refresh reliability, chart usability

## Deliverables
- Prioritized feedback backlog items with user quotes and frequency counts
- Monthly sentiment summary
- Feature request rankings with estimated user impact
